# EasySite — 프로젝트 마스터 문서

> 이 문서는 Claude Code가 작업을 시작하기 전에 반드시 읽어야 하는 프로젝트 전체 설계서입니다.
> 모든 코드 작업은 이 문서의 원칙과 구조를 따라야 합니다.

---

## 1. 프로젝트 개요

### 1-1. 서비스 한 줄 정의

> "아이디어만 있으면 됩니다. 나머지는 AI가 채워드립니다."

웹사이트 제작이 필요한 사장님들이 정보를 잘 모르고 아이디어만 있어도, AI 자동화로 견적부터 시안·정보 수집·제작 진행 추적까지 셀프 서비스로 경험할 수 있게 하는 **1인 프리미엄 웹사이트 제작 플랫폼**.

### 1-2. 타겟

- 모든 사장님 (1인 사업자, 소상공인, 스타트업, 중소기업)
- 나이대 폭넓음 (20대~60대). **UI/UX는 60대도 이해할 수 있는 직관성 필수**
- 공통 특징: "사이트 만들고 싶은데 뭐가 필요한지, 뭘 준비해야 하는지 모름"

### 1-3. 시장 포지셔닝

| 카테고리 | 예시 | 한계 |
|---|---|---|
| AI 빌더 (DIY) | Wix, 식스샵, Gamma | 사장님이 직접 다 만들어야 함 |
| 외주 중개 | 숨고, 크몽, 위시켓 | 매칭만 함. 제작 안 함 |
| 에이전시 | 닷클, 넥스트티 | 상담 후 견적, 자동화 약함 |
| **EasySite (우리)** | — | **AI가 전 과정 자동화, 사람은 제작만** |

### 1-4. 핵심 차별화 (랜딩 카피의 4가지 축)

1. **24시간 안에 견적 + 미리보기** — 다른 곳은 2주 걸림, 우리는 즉시
2. **글 쓸 필요 없음** — AI가 카피·메뉴·구조 초안 자동 생성
3. **보면서 직접 수정 요청** — 카톡 핑퐁 없이 사이트 위에서 코멘트
4. **가격 공개** — 사장님이 가장 답답해하는 부분

## 3. 기술 스택

### 3-1. 확정된 스택

- **Frontend/Backend**: Next.js 14+ (App Router, TypeScript)
- **Styling**: Tailwind CSS
- **DB**: Supabase (PostgreSQL + Auth + Storage)
- **AI**: Anthropic API (Claude Sonnet 모델)
- **결제**: 토스페이먼츠 (Phase 2부터)
- **이메일**: Resend
- **알림 (본인용)**: 텔레그램 봇
- **알림톡 (고객용)**: 카카오 알림톡 (Phase 5)
- **호스팅**: Vercel
- **도메인**: easysite.kr (구매 예정)
- **개발 환경**: GitHub Codespaces + Claude Code in terminal

### 3-2. 코드 스타일 원칙

- 모든 UI 텍스트는 한국어
- 컴포넌트는 함수형 + Hooks
- 상태 관리는 React `useState` / `useReducer` 기본, 필요시 Zustand 검토
- 폼은 `react-hook-form` + `zod`

### 3-3. 톤 가이드 (UI 문구)

모든 UI 문구는 **"정보를 잘 모르는 사장님"**을 배려해야 함.

| ❌ 나쁨 | ✅ 좋음 |
|---|---|
| "페이지 수를 입력하세요" | "몇 개 정도 페이지가 필요할까요? (잘 모르시겠으면 추천받기 눌러주세요)" |
| "회사 소개 문구를 작성해주세요" | "회사에 대해 한 줄만 알려주세요. 자세한 소개는 AI가 초안 만들어드려요." |
| "결제 기능 필요 여부" | "고객이 사이트에서 결제하나요? 예) 상품 판매, 강의 등록" |

모든 질문에 **"잘 모르겠어요"** 옵션 + **"이게 뭐예요?"** 툴팁 제공.

## 4. 5단계 로드맵 (전체 16주)

### Phase 1 (4주): 입구 — "쉽게 문의"

- **목표**: 견적 위저드 + AI 챗봇 + 벤치마크 추천 + 경쟁사 분석 + AI 카피 초안
- **검증 지표**: 위저드 링크 클릭 30건 / 완료율 40% / 계약 1~2건

### Phase 2 (4주): 결제 직후 — "WOW 모먼트"

- **목표**: 3분 데모 자동 생성 + 컬러 팔레트 자동 추천 + 목업 자동 생성

### Phase 3 (3주): 제작 중 — "안심 시스템"

- **목표**: 진행률 대시보드 + 체크리스트 + 단계별 만족도 체크 + 댓글 위치 지정

### Phase 4 (3주): 완료 후 — "관계 유지"

- **목표**: 자동 후기 요청 + 포트폴리오 자동 추가 + 재방문 고객 인식 + 회원가입

### Phase 5 (2주): 외부 연동 — "어려운 거"

- **목표**: 인스타·네이버 플레이스 크롤링 + 카카오 알림톡 + 다국어 견적

## 작업 시 규칙 (Claude Code 전용)

1. 작업 시작 전 이 문서 전체를 다시 읽고 시작한다.
