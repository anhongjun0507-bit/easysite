# /letters 제목·UI 서체 라이선스

`scripts/subset-letters-font.py` 가 원본을 받아 서브셋한 결과물이다.
세 폰트 모두 OFL 계열이라 수정(서브셋)·재배포·웹 임베딩이 허용된다.

## 배민 주아 (Jua) (`jua`)

- 원본: https://raw.githubusercontent.com/google/fonts/main/ofl/jua/Jua-Regular.ttf
- 라이선스: SIL Open Font License 1.1 — https://fonts.google.com/specimen/Jua

## 카페24 써라운드 (`ssurround`)

- 원본: https://raw.githubusercontent.com/fonts-archive/Cafe24Ssurround/main/Cafe24Ssurround.ttf
- 라이선스: OFL (수정·재배포 가능, 폰트 유료 판매 금지) — https://fonts.cafe24.com

## 카페24 동동 (`dongdong`)

- 원본: https://raw.githubusercontent.com/fonts-archive/Cafe24Dongdong/main/Cafe24Dongdong.ttf
- 라이선스: OFL (수정·재배포 가능, 폰트 유료 판매 금지) — https://fonts.cafe24.com

## 제외한 후보

- 넥슨 배찌체 — "폰트 파일의 수정·복제·배포 금지, 배포된 형태 그대로 사용" 조항이라
  서브셋(수정 + 재배포) 파이프라인에 쓸 수 없다.
